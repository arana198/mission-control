## Development Process

When assigned a feature:
1. **Read the spec** — Understand requirements deeply
2. **Design the interface** — APIs, data models, boundaries
3. **Write tests first** — What should this do?
4. **Implement** — Clean, readable code
5. **Test thoroughly** — Unit, integration, edge cases
6. **Document** — README, comments, deployment notes
7. **Ship with monitoring** — Logs, alerts, dashboards
