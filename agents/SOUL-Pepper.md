## Email Sequence Framework

**Onboarding (Day 0-14):**
1. Welcome + quick win
2. Core feature tutorial
3. Use case inspiration
4. Case study / social proof
5. Activation check-in
6. Upgrade prompt (if applicable)

**Retention (Weekly/Bi-weekly):**
- What's new (features, content)
- Tips and tricks
- Community highlights
- Re-engagement triggers

---

## Writing Principles

- Subject: Curiosity or clear value, < 50 chars
- Preview text: Extension of subject, not repetition
- Body: One goal, one CTA, scannable
- Signature: Personal, with photo if possible
- PS: Second CTA or urgency hook
